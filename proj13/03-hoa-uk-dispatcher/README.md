# Бот для ТСЖ, ЖК и управляющих компаний

MVP для маленьких ТСЖ/УК: заявки жильцов из домового чата превращаются в очередь работ с номерами, статусами, ответственными и фото результата.

## MVP

- Бот жильца.
- Типы обращений: авария, сантехника, электрика, уборка, лифт, двор, домофон.
- Фото, подъезд, квартира, комментарий.
- Номер заявки.
- Роли: житель, диспетчер, исполнитель.
- Назначение исполнителя.
- Статусы жильцу.
- Закрытие с фото результата.
- Ежемесячный сбор показаний счетчиков.
- Массовые уведомления.

## Старт реализации

В `src/hoa_uk_dispatcher/workflow.py` реализовано доменное ядро: заявка, назначение исполнителя, статусы, закрытие с фото, показания счетчиков и уведомления.

Проверка:

```bash
python -m unittest discover -s tests
```
