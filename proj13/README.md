# proj13

Рабочая папка для трех отдельных MVP-проектов ботов:

1. `01-field-master-dispatcher` - Telegram-first диспетчер выездных мастеров.
2. `02-postservice-clinic-care` - постсервисное сопровождение пациентов/клиентов клиник.
3. `03-hoa-uk-dispatcher` - заявки жильцов для ТСЖ, ЖК и управляющих компаний.

## Как вести работу

Каждый проект содержит одинаковый контур управления:

- `reviews/` - ревью идеи, рисков, MVP и рынка.
- `goals/` - цели проекта, каждая цель оформлена отдельным файлом.
- `prompts/` - промпты, которыми можно запускать реализацию целей.
- `HANDOFF.md` - текущее состояние, что сделано, что делать дальше.
- `src/` и `tests/` - стартовая реализация доменной логики.
