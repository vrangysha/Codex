# Постсервисный бот для клиник

MVP для стоматологий, косметологий и ветклиник: сопровождение после процедуры, контроль состояния, эскалация тревожных симптомов, отзывы и повторные визиты.

## Важное ограничение

Бот не лечит, не ставит диагноз и не заменяет врача. Он отправляет утвержденные клиникой памятки, собирает состояние и передает тревожные случаи администратору/врачу.

## MVP

- Шаблоны протоколов по услугам.
- Расписание сообщений 6/24/72 часа.
- Анкеты состояния.
- Загрузка фото.
- Красные флаги и эскалация.
- Напоминание о контрольном визите.
- Сбор отзыва.

## Старт реализации

В `src/postservice_clinic_care/workflow.py` реализовано доменное ядро: кейс наблюдения, расписание, чек-ин, красные флаги и эскалация.

Проверка:

```bash
python -m unittest discover -s tests
```
