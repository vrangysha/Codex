# Goal 002: Protocol Catalog

Status: Ready

## Outcome

Создан каталог протоколов по услугам: памятка, расписание, вопросы, красные флаги, правила контрольного визита и отзыва.

## Acceptance Criteria

- Протоколы можно хранить в читаемом формате.
- Админ может выбрать протокол при создании кейса.
- Тексты отделены от логики бота.
