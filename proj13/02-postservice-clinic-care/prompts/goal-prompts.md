# Goal Prompts

## Goal 001

Реализуй доменное ядро постсервисного бота для клиник: кейс наблюдения, утвержденная памятка, расписание чек-инов, анкета состояния, красные флаги и эскалация. Запрещено генерировать медицинские советы. Добавь unittest-тесты.

## Goal 002

Создай каталог протоколов по услугам для стоматологии, косметологии и ветклиники. Вынеси тексты памяток, вопросы, расписание и красные флаги в конфиг, который можно редактировать без изменения кода.

## Goal 003

Сделай минимальную панель эскалаций: список тревожных кейсов, карточка пациента/питомца, фото, история чек-инов, статусы реакции администратора и журнал действий.

## Goal 004

Добавь планировщик чек-инов и message adapter для постсервисного бота. Планировщик должен находить due-сообщения по расписанию кейса, adapter должен отправлять только утвержденные тексты из протокола, принимать ответы анкеты, вызывать evaluate_checkin и создавать EscalationTicket при красных флагах. Добавь unittest-тесты.

## Goal 005

Собери delivery package для постсервисного бота клиник: demo entrypoint, runbook внедрения, integration contract для МИС/CRM/мессенджеров, README с медицинскими ограничениями и состоянием MVP. Бот не должен генерировать лечение или диагнозы.
