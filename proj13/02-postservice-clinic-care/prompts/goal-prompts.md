# Goal Prompts

## Goal 001

Реализуй доменное ядро постсервисного бота для клиник: кейс наблюдения, утвержденная памятка, расписание чек-инов, анкета состояния, красные флаги и эскалация. Запрещено генерировать медицинские советы. Добавь unittest-тесты.

## Goal 002

Создай каталог протоколов по услугам для стоматологии, косметологии и ветклиники. Вынеси тексты памяток, вопросы, расписание и красные флаги в конфиг, который можно редактировать без изменения кода.

## Goal 003

Сделай минимальную панель эскалаций: список тревожных кейсов, карточка пациента/питомца, фото, история чек-инов, статусы реакции администратора и журнал действий.
