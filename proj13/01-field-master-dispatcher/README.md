# Бот-диспетчер для выездных мастеров

Telegram-first MVP для маленьких сервисных команд: ремонт техники, кондиционеры, сантехника, электрика, окна, двери, мебель, сервисные центры.

## MVP

- Прием заявки: услуга, адрес, фото/видео, описание.
- Уточнение деталей: модель, срочность, время, район.
- Предварительный диапазон цены или отправка мастеру на оценку.
- Карточка заявки для администратора.
- Назначение мастера.
- Статусы для клиента.
- Отзыв, фото результата, напоминание о гарантии.

## Старт реализации

В `src/field_master_dispatcher/workflow.py` реализовано доменное ядро: заявка, статусы, назначение мастера, смена статуса, карточка администратора и экспорт строки.
В `src/field_master_dispatcher/intake.py` реализован framework-agnostic intake flow: пошаговый сбор заявки, сериализация черновика и создание доменной заявки.

Проверка:

```bash
python -m unittest discover -s tests
```
