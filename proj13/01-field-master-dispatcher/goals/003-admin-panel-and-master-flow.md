# Goal 003: Admin Panel And Master Flow

Status: Ready

## Outcome

Администратор видит очередь заявок, назначает мастера, мастер получает карточку и меняет статусы.

## Acceptance Criteria

- Есть список заявок с фильтрами по статусу и району.
- Назначение мастера меняет статус и создает уведомление.
- Клиент получает статусы `принято`, `мастер назначен`, `едет`, `выполнено`.
