# Кибер Арена — пакет ассетов

Пакет подготовлен для верстки лендинга компьютерного клуба по утвержденному визуальному макету.

## Структура

- `branding/` — логотипы SVG/PNG, знак, favicon и OG-cover.
- `images/hero/` — desktop/mobile hero-фоны и исходный референс.
- `images/zones/` — четыре изображения игровых зон.
- `images/tournaments/` — очищенные фоны турниров и референсные карточки.
- `images/gallery/` — пять интерьерных изображений.
- `images/avatars/` — три аватара отзывов с прозрачностью.
- `images/map/` — карта с маркером и чистый абстрактный map-background без маркера.
- `images/cta/` — чистый CTA-фон и отдельный игровой декор.
- `icons/` — отдельные SVG-иконки и общий SVG-sprite.
- `patterns/` — сетка, шум, свечения и неоновая рамка.
- `tokens/` — CSS/JSON design tokens и палитра для Tailwind.
- `reference/` — полный макет и отдельные секции для сверки верстки.

## Форматы

Растровые фотографии экспортированы в WebP высокого качества; основные ключевые фоны также продублированы в PNG. Логотипы, иконки и паттерны — SVG, поэтому масштабируются без потери качества.

## Важное ограничение

Исходником был единый растровый макет 724×2172 px, а не Figma-файл с отдельными слоями. Поэтому фотографии были аккуратно извлечены, реконструированы и увеличены; это не исходные full-resolution рендеры. Для production они пригодны как веб-ассеты, но при печати или экстремальном увеличении потребуется повторная генерация исходных сцен.

## Подключение SVG sprite

```html
<svg class="icon" aria-hidden="true">
  <use href="/assets/icons/icon-sprite.svg#icon-gamepad"></use>
</svg>
```

## Цвета

Основные переменные находятся в `tokens/design-tokens.css`. Рекомендуемый фон страницы — `#0B0F19`, основной CTA — градиент `#FF2BD6 → #8A5CFF`, информационный акцент — `#00D9FF`.
